Artwork

For the course titled Aerospace Human Factors Engineering, I worked with within a team of four to create an interface to monitor and send commands to a CubeSat. The display was created for the general CubeSat, but will be personalized to Columbia’s CubeSat Mission design and payload.

We testing the display using the modified Cooper Harper scale to evaluate the various qualities including integrated vs separate displays, touch screen vs typed pseudocode input, and the overall effectiveness of the display design.

The display includes data on satellite location, orientation, telemetry data, and temperature, as well command window users may interface with. The menu can be customized to present more detailed information on a desired aspect.
